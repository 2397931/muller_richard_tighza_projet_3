# muller_richard_tighza_projet_3
Réalité Mixte

## Description du jeu
Un jeu d'évasion en VR où tu progresses dans un donjon sombre avec de l'équipement limités, comme un briquet, gagnes de l'argent pour améliorer ton équipement dans une boutique ( par exemple en achetant une lampe de poche), dans le but d'amasser suffisamment d'argent pour ouvrir un coffre contenant la clé qui te permettra de t'échapper.


## Notre proposition
Tu débute le jeu dans une des salles du donjon avec pour seul outil un briquet qui te permet de te naviguer dans l'obscurité. En explorant les différentes pièces, tu dois surmonter des obstacles et trouver des moyens de gagner de l'argent. À force de progresser, tu peux retourner à la boutique pour améliorer ton équpement.

### But principale du jeu
Pour votre quête de liberté.,vous devez amasser de l'argent pour améliorer vos équipements. 

#### Idées optionelles:
Si vous avez besoin d'un petit indice, chaque pièce a ses sons uniques liés à un objet spécial et vous devez réussir de petits défis afin d'obtenir un bonus. Ce bonus vous donnera un indice sur l'emplacement de la porte.
